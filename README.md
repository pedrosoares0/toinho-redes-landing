> **A estética premium do esporte em uma interface fluida e de alta performance.**

Este projeto foi desenvolvido para ser o cartão de visitas digital da **Toinho Redes**, empresa líder em redes esportivas de alta qualidade na Bahia. O foco principal foi unir a robustez do produto com uma experiência de navegação **moderna, rápida e visualmente impactante**, utilizando tendências atuais de UI Design como *Glassmorphism* e *Bento Grids*.

A interface destaca a durabilidade e o alcance das redes Toinho, facilitando o agendamento de orçamentos via WhatsApp com CTAs estratégicos e um catálogo interativo.

---

## ✨ Destaques do Projeto

*   **Glass-Glossy UI:** Implementação de efeitos de vidro e brilho premium através de gradientes complexos e pseudo-elementos (`::before`, `::after`), garantindo um visual "high-end".
*   **Bento Grid Architecture:** Organização modular de estatísticas e diferenciais, otimizando o espaço e a hierarquia visual.
*   **Experiência Mobile-First:** Arquitetura pensada 100% para o uso em smartphones, com navegação fluida e elementos de toque otimizados.
*   **Interações Avançadas:**
    *   **3D Tilt Effect:** Cards de estatísticas que reagem ao movimento do mouse.
    *   **Apple-Style Slider:** Catálogo de produtos com scroll horizontal suave e barra de progresso dinâmica.
    *   **Custom Lightbox:** Galeria de portfólio integrada para visualização imersiva de projetos.
    *   **Scroll Reveal:** Animações de entrada coordenadas via *Intersection Observer API*.

## 🛠️ Stack Técnica (Para Desenvolvedores)

Este projeto foi construído "do zero" (Vanilla), focando em performance bruta, semântica rigorosa e facilidade de manutenção sem dependências externas pesadas.

*   **HTML5:** Estrutura semântica focada em SEO e acessibilidade.
*   **CSS3 (Modern Features):**
    *   Uso extensivo de **CSS Variables** para gerenciamento de cores e temas.
    *   Layouts complexos com **Flexbox** e **CSS Grid**.
    *   Animações nativas e transições otimizadas para GPU.
*   **JavaScript (ES6+):**
    *   Lógica de slider customizada.
    *   Manipulação de DOM para efeitos de paralaxe e rotação de texto.
    *   Otimização de performance via *Intersection Observer*.

    ```bash
    cd toinho-redes-landing
    ```
