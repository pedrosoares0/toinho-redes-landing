> **A estética do esporte em uma interface fluida e de alta performance.**

Landing page desenvolvida para a **Toinho Redes**, focada em unir design moderno (*Glassmorphism* e *Bento Grids*) com uma experiência de conversão direta via WhatsApp.

---

## ✨ Destaques

*   **Glass UI:** Efeitos de vidro e brilho premium com CSS puro.
*   **Bento Grid:** Organização modular e moderna de conteúdos.
*   **Mobile-First:** Experiência otimizada para smartphones.
*   **Interações:** Slider estilo Apple, 3D tilt nos cards e animações de scroll.

## 🛠️ Tecnologias

*   **HTML5 & CSS3:** Semântica e variáveis CSS para layouts complexos.
*   **Vanilla JavaScript:** Slider customizado e animações via *Intersection Observer*.
